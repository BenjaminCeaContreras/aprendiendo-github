Procedimiento

Conceptos clave
CONVOLUCIÓN: La convolución es una operación matemática que describe cómo una señal se transforma al pasar por un sistema. En acústica, ese “sistema” puede ser una sala, un micrófono, un altavoz o cualquier medio que afecte el sonido.

Ejemplo: Tienes un sonido original (por ejemplo, una palmada). Ese sonido entra en un espacio (una iglesia, una habitación pequeña, un túnel). El espacio añade ecos, reverberaciones y cambios de timbre. La convolución es la forma matemática de combinar el sonido original con la “huella acústica” del espacio.

ISO 3382-2 sección 2
La ISO 3382-2 sección 2 asegura que cuando alguien diga “esta sala tiene un tiempo de reverberación de 0,7 segundos”, ese valor se haya obtenido con un método estandarizado, usando equipo y condiciones comparables. Es la base para diseñar y evaluar espacios donde la inteligibilidad del habla es lo más importante.

Procedimiento según la norma
1. Generar el impulso acústico con una fuente omnidireccional (globo).
2. Registrar la respuesta con micrófonos calibrados en varias posiciones.
3. Procesar digitalmente la señal para obtener la curva de decaimiento.
4. Calcular parámetros aplicando fórmulas estandarizadas.
5. Reportar resultados en tablas y gráficas, siguiendo el formato ISO.
Ubicaciones y fuentes a estudiar
Para este caso se realizó la respuesta impulso (RI) de tres salas distintas, siendo el primer lugar una sala reverberante, el segundo el hall de la sala reverberante, y el ultimo, el laboratorio del instituto de acústica. Tres lugares con características y fines distintos. Previo a esto se grabaron 3 audios distintos en una sala anecoica, una voz, un instrumento, y la vos e instrumentos juntos. Estos audios serian los puestos a prueba a ser convolucionados.

Procesamiento y análisis de datos
Como bien se comentó al principio, al momento de realizar las RI se realizaron varias grabaciones de las cuales solo fue seleccionada una en estos casos para cada sala. Posteriormente lo primero que se realizó fue un etiquetado de audios para un mayor orden visual y velocidad de trabajo. Después del etiquetado procedemos a realizar la obtención de las señales convolucionadas y sus gráficos correspondientes, obteniendo un detalle gráfico y auditivo de cada señal trabajada mediante un código en Python.
